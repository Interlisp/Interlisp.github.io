--- defaults: - '""' flags: - cmap - svg minimums: [] name: tooltip types: - escString used_by: NEC --- Tooltip annotation attached to the node or edge. If unset, Graphviz will use the object's label if defined. Note that if the label is a record specification or an HTML-like label, the resulting tooltip may be unhelpful. In this case, if tooltips will be generated, the user should set a tooltip attribute explicitly.
