--- defaults: - 8(output) - 20(overlap and image maps) flags: [] minimums: [] name: samplepoints types: - int used_by: N --- If the input graph defines the vertices attribute, and output is dot or xdot, this gives the number of points used for a node whose shape is a circle or ellipse.
