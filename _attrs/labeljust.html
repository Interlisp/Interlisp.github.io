--- defaults: - '"c"' flags: [] minimums: [] name: labeljust types: - string used_by: GC --- Justification for cluster labels. If "r", the label is right-justified within bounding rectangle; if "l", left-justified; else the label is centered. Note that a subgraph inherits attributes from its parent. Thus, if the root graph sets labeljust to "l", the subgraph inherits this value.
