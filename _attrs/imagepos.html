--- defaults: - '"mc"' flags: [] minimums: - '' name: imagepos types: - string used_by: N --- Attribute controlling how an image is positioned within its containing node. This only has an effect when the image is smaller than the containing node. The default is to be centered both horizontally and vertically. Valid values:
tlTop Left
tcTop Centered
trTop Right
mlMiddle Left
mcMiddle Centered (the default)
mrMiddle Right
blBottom Left
bcBottom Centered
brBottom Right
