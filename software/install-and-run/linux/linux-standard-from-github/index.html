Download WSL (Standard Installation) Install Files from Github

  1. Go to the Releases page for Interlisp/Medley on github.com. Here you will see a list of releases in reverse chronological order.

 

  1. Choose the release you are interested in (most likely the release marked Latest) and click on the arrow next the word Assets to reveal the list of assets (i.e., installation files) for that release.

  2. To download the .tgz install file, click on the link that ends with .deb and begins with either medley-full-linux-x86_64 (if you are installing on an x86_64-based machine) or medley-full-linux-aarch64 (if you are installing on an Arm64-based machine) or medley-full-linux-arm7l (if you are installing on an Arm7-based machine - e.g., a Raspberry Pi).
