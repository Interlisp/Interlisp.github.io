If defined, `labelURL` is the link used for the label of an edge.

`labelURL` overrides any [`URL`]({{< ref "URL.md" >}}) defined for the edge.

See also:

- [`edgehref`]({{< ref "edgehref.md" >}}), [`edgeURL`]({{< ref "edgeURL.md" >}})
